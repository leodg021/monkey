# Monkey Language Interpreter and Compiler

Project based on the books by **Thorsten Ball** on writing an interpreter and compiler for the _Monkey_ programming language using _Go_.

Further documentation and the developing progress will be here annotated in the nearby future.


## References

\[1\] T. Ball, [_Writing an Interpreter in Go_](https://interpreterbook.com/). 2016.

\[2\] T. Ball, [_Writing a Compiler in Go_](https://compilerbook.com/). 2018.
